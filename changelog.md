# Diyar DVR phase25 v5.18

Реализовано:
* Новый информационный ряд с увеличенными иконками, без слова REC.
* Разрешение меняется по режиму: мозаика — 1920×1080, отдельные камеры — 1280×720.
* Новые иконки настроек, архива, автора, USB, FPS, битрейта, часов, камер, цикла и микрофона.
* USB недоступен и отображается серым, пока флешка не подключена.
* Кнопка СТАРТ блокируется, если не выбрано доступное хранилище. СТОП остаётся доступным во время записи.
* Добавлена запись звука AAC 44,1 кГц непосредственно в MP4 для обоих режимов.
* Добавлен запрос разрешения микрофона Android.
* Настройка звука сохраняется после перезагрузки.
* Добавлены русские и казахские надписи состояния микрофона.
* Исправлена серверная проверка хранилища — запись также не запустится обходным способом без выбранной памяти.
